Installing RandGraph

Prerequisites

  • The ghc Haskell compiler, version 6.4 or newer. Current hugs almost works too, but does not support cpp preprocessing which we use in several places.
  • Json library, built in json/. See INSTALL.
  • SashoLib.hs and MonadRandom.hs, which are in lib/haskell/
  • Some components from the Faerieplay compiler in sfdl-compiler/
  • The Haskell functional graph library (in the Data.Graph.Inductive package). TODO url.

Building

$ make install
Builds and installs into the dist directory configured in the main config file pir/config.make.
